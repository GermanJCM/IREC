# 🔐 Plan Maestro de Ciberseguridad - IREC

## 📌 Descripción
Este proyecto consiste en una **página web desarrollada para presentar un Plan Maestro de Ciberseguridad**, realizado como parte de un trabajo universitario.

El objetivo fue colaborar con una organización real (**OSSAPAT**) para diseñar un plan completo que incluyera:

- Evaluación de riesgos  
- Análisis de vulnerabilidades  
- Pruebas de penetración (pentesting)  
- Estrategias de seguridad  

La página web funciona como medio de presentación del proyecto, mostrando la información de manera clara y visual.

---
